                MYHILL-NERODE VISUALIZER
              

              

                
Distinguishable string classes for L = {aⁿbⁿ}

                

                

                  
Infinitely many equivalence classes

                  
Each class needs a distinct DFA state

                  
No finite DFA can exist

                  
∴ Language is NOT regular

                

                

                  
                  
                

              

            

          


          
          

            

              

                

                CLOSURE PROPERTY PROOF
              

              

                
Proof by Contradiction via Intersection

                

                  
Assume L is regular

                  

                  
Intersect with a*b* (a regular language)

                  

                  
Result: L ∩ a*b* = {aⁿbⁿ}

                  

                  
Regular languages are closed under intersection

                  

                  
{aⁿbⁿ} must also be regular — but it's NOT (Pumping Lemma)

                  

                  
CONTRADICTION → L is NOT regular ∎

                

              

            

          


        

      


      
      

        

          

          STEP-BY-STEP PROOF TIMELINE
        

        

          

            

              

              
Step 1 — Assume L is regular

              
Suppose for contradiction that L is a regular language.

            

            

              

              
Step 2 — Pumping length exists

              
By the Pumping Lemma, there exists a pumping length p such that every string of length ≥ p can be pumped.

            

            

              

              
Step 3 — Choose string s

              
Choose s with |s| ≥ p. We know s ∈ L.

            

            

              

              
Step 4 — Split s = xyz

              
By the Pumping Lemma, s can be split as s = xyz where:

                |xy| ≤ p  
                |y| ≥ 1  
                x=ε, y=aaaa, z=bbbb
              

            

            

              

              
Step 5 — Pump y

              
Consider xyⁱz for all i ≥ 0. The Pumping Lemma guarantees xyⁱz ∈ L for all i.

            

            

              

              
Step 6 — Find contradiction

              
When i=0 (or i=2), the pumped string leaves L — the number of a's and b's becomes unequal.

            

            

              

              
Conclusion — L is NOT regular ∎

              
Contradiction! xyⁱz ∉ L for some i, violating the Pumping Lemma. Therefore L is not a regular language.
